# JavaScript Mini Projects

This repository contains basic JavaScript DOM projects created for practice.

## Projects Included

### 1. Stopwatch
- Start, Stop (Pause), and Reset functionality
- Time tracking in minutes, seconds, and milliseconds
- Built using setInterval and DOM manipulation

### 2. Image Swap
- Click on a small image to swap it with the main image
- Uses simple JavaScript function and DOM handling

## Tech Used
- HTML
- CSS
- JavaScript

## Purpose
These projects are created to practice JavaScript concepts like:
- DOM manipulation
- Event handling
- Timing functions
